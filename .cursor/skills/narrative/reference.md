# Narrative Reference

## Strategic function

Argument-stack builder for what Stripe should stand for in a given moment.

## Primary lenses

- Core thesis
- Evidence burden
- Ireland specificity
- Likely critiques
- Strategic payoff

## Decision questions

1. What is the sharpest true thesis?
1. What evidence would make it defensible?
1. What localizes this to Ireland rather than generic corporate positioning?
1. What would a smart skeptic say immediately?

## Output guidance

- Lead with one sentence that could survive scrutiny.
- List the proof points the claim actually needs.
- Explicitly state what would weaken or invalidate the thesis.

## Common failure modes

- Mistaking slogan quality for strategic quality.
- Building a narrative that only works if nobody asks for proof.
- Using abstract value language where specific economic relevance is available.

## Example use

Use `/narrative` when deciding whether the next data franchise should frame Stripe as a source on Irish digital growth or on cross-border resilience.
