# Narrative

## Mission

Find the sharpest true thesis, then expose what evidence it actually needs.

## Use this skill when

- the user explicitly asks for `/narrative` or the equivalent workflow
- the operating context matches this skill's mission more closely than a generic drafting pass
- the result needs to be stored or reasoned about as part of the Comms-Stack system

## Read before acting

- `knowledge/stripe-narrative.md`
- relevant campaign archive entries in `knowledge/campaign-archive/`
- `knowledge/policy-landscape.md` if the topic intersects regulation

## Workflow

1. State the core claim in one sentence.
2. Break it into supporting arguments that can be evidenced.
3. List what would make the thesis stronger or weaker.
4. Identify where the Ireland angle is specific rather than generic.
5. Define the critiques a smart skeptic would raise.

## Output contract

Use the relevant repo template when it helps. The output should include:

- one-sentence thesis
- supporting arguments
- proof points needed
- likely critiques
- Ireland resonance
- recommendation

## Quality bar

- Reject vague corporate framing.
- Prefer fewer stronger arguments to many soft ones.
- If the thesis only works with hype, kill it.
- Say what evidence is missing.

## Escalate immediately when

- claim depends on unavailable evidence
- topic is policy-sensitive
- global and local framing meaningfully diverge

## Memory update rule

Update `knowledge/stripe-narrative.md` only when the thesis changes durable positioning, not every campaign nuance.

## Do not

- Do not confuse slogan-writing with strategy.
- Do not smuggle weak proof points into the core claim.
- Do not use abstraction when a local fact would do.
