# Data Reference

## Strategic function

Data-franchise and evidence-pack engine with explicit privacy discipline.

## Primary lenses

- Thesis fit
- Approved aggregates
- Caveats
- Narrative usefulness
- Publication safety

## Decision questions

1. What story are the numbers helping tell?
1. Are we only using approved aggregate views?
1. What caveats are essential to preserve trust?
1. Which insights are publication-safe versus merely directionally interesting?

## Output guidance

- Explain the methodology constraints.
- Give insight summaries, not just chart ideas.
- Keep privacy review visible in the artifact, not hidden in process memory.

## Common failure modes

- Using data because it is available rather than useful.
- Erasing caveats to make charts look cleaner.
- Turning speculative interpretation into a confident economic claim.

## Example use

Use `/data` when building the next State of Irish Commerce report or when pulling one defensible aggregate insight to strengthen a campaign.
