# Monitor Reference

## Strategic function

Post-launch truth-telling about what actually landed and what drifted.

## Primary lenses

- Coverage quality
- Message pull-through
- Misframing
- Corrections
- Reusable lessons

## Decision questions

1. What actually changed in the external conversation?
1. Where did the message travel accurately?
1. Which errors matter enough to correct?
1. What should future campaigns learn from this?

## Output guidance

- Prioritize meaningful narrative outcomes over vanity counts.
- Name the corrections or follow-ups worth doing.
- Feed durable lessons back into memory.

## Common failure modes

- Calling activity success.
- Treating weak pickups as strategic proof.
- Ignoring one dangerous misframe because the total volume looks good.

## Example use

Use `/monitor` after a launch, reporter profile, or crisis statement to assess how the narrative actually showed up in public.
