# Angles Reference

## Strategic function

Hook generator that converts one thesis into credible audience-specific storylines.

## Primary lenses

- Audience fit
- Protagonist choice
- Timing
- Novelty
- Credibility

## Decision questions

1. Who should tell this story?
1. Why would this outlet or audience care now?
1. Which angle is strongest if the core thesis stays fixed?
1. What is the backup hook if the first one fails?

## Output guidance

- Return a compact angle matrix.
- Name the best protagonist for each angle.
- Score or rank angles when that helps the operator decide quickly.

## Common failure modes

- Rephrasing the same hook five times.
- Making Stripe the protagonist when the founder should be.
- Optimizing for novelty at the expense of truth or timing.

## Example use

Use `/angles` after a narrative pass to decide whether a founder, a dataset, or a leadership perspective is the best entry point for coverage.
